8 更多 API 端点

除 Chat 对话外，还支持以下 OpenAI 兼容端点：

POST /v1/embeddings — 文本向量化

POST /v1/images/generations — 图片生成

POST /v1/images/edits — 图片编辑

POST /v1/images/variations — 图片变体

POST /v1/audio/speech — TTS 语音合成

POST /v1/audio/transcriptions — 语音转写

POST /v1/audio/translations — 语音翻译

POST /v1/videos/generations — 视频生成

POST /v1/moderations — 内容审核

GET /v1/models — 模型列表

所有端点均支持计费（按 token / 按张 / 按音频大小），需上游 API 支持对应能力。

9 计费说明

Chat/Completions：本地 tiktoken 计算 prompt + 上游 usage 计算 completion（含思考模型的 reasoning_tokens）

Embeddings：优先上游 usage，fallback tiktoken 估算

Images：按张数估算（每张约 1000 tokens）

Audio TTS：按音频大小估算

Audio 转写：优先上游 usage，fallback tiktoken

Video：优先上游 usage，fallback 估算

额度批量扣费（5秒间隔），关闭服务时自动结算剩余。
